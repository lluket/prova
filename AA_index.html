AKBE, lo que quieras en el lugar ideal.

Para explicar lo que hacemos con más facilidad, lo haremos a partir de un proyecto donde trabajamos en la ubicación de un resort deportivo.

A primera instancia, puede parecer sencillo ubicar una instalación de este tipo, pero en nuestro caso la voluntad es encontrar una zona ideal en la que implantar instalaciones deportivas cerca de núcleos de población, y no en el centro de un núcleo. Al mismo tiempo, ha de ser un lugar tranquilo y en contacto con la naturaleza. De esta manera se conseguirá que sea un lugar atractivo para los clientes para hospedarse y realizar actividades deportivas. Con estos criterios, uno de los mayores problemas que se han de resolver es que no sea una zona con riesgo. Un resort deportivo es una instalación que ofrece actividades recreativas a las personas que se hospedan allí. Es un elemento muy atractivo para el turismo, ya que ofrece un entorno perfecto para realizar diferentes actividades. El resort que se quiere ubicar, a diferencia de los habituales, se construirá cerca de masas de agua para que se puedan realizar actividades como piragüismo y cerca de zonas verdes para poder realizar senderismo, entre otros.

El objectivo principal es localizar el punto óptimo para ubicar un resort deportivo en la comarca del Tarragonès a partir de un SIG.

También tenemos una serie de objetivos específicos que ayudan a definir lo que se quiere lograr:
  1. Seleccionar las zonas que sean más accesibles para la población con la finalidad de que tenga el mayor público posible, que este a proximidad de las zonas urbanas.
  2. Localizar una zona accesible para otros núcleos de población.
  3. Se debe seleccionar un sitio tranquilo que esté apartado de las carreteras y que esté cerca de un espacio protegido.
  4. Ubicar el resort en la proximidad de un área verde y zonas de agua para que los usuarios tengan buenas vistas desde el resort y puedan realizar actividades en estos.
  5. Escoger una zona donde haya espacio suficiente en caso de que se quiera ampliar el resort.
  6. Localizar una zona sin riesgo de inundación.
